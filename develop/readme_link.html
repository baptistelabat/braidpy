BraidPyïƒ

A Python library for working with braids, with the goal to help design real world complex 3D braids, using recent advances in braid theory.

Featuresïƒ

  • [x] Mathematical representation of braids with braid words using Artinâ€™s generator

  • [x] Conversion between different braidword formats

  • [x] Braid operations and manipulations

  • [x] ASCI, 2D and 3D visualization capabilities

  • [x] Computations of mathematical properties of braidword or invariants of braid

  • [ ] Annulus braids

  • [ ] Simulation of machine braiding process

Basic exampleïƒ

from braidpy import Braid

# Create a braid
b = Braid([1, 2, -1])

# Perform operations
result = b * b.inverse()

Release are available on Pypi and it should be very easy to install braidpy in your python environment.

Using pip

pip install braidpy

Or using poetry

poetry add braidpy

With uv (package manager used on the project)

uv add braidpy
